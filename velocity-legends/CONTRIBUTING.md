# Contributing to Velocity Legends

Thank you for your interest in contributing! This is a UI showcase project for a racing game.

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

## Areas for Contribution

- 🖼️ **Image Assets**: Replace Google-hosted images with local/royalty-free alternatives
- 🎨 **New Screens**: Add additional game screens (settings, shop, multiplayer lobby)
- 📱 **Responsive Improvements**: Better mobile and tablet support
- ⚡ **Performance**: Optimize animations and reduce bundle size
- 🌐 **Accessibility**: Improve keyboard navigation and screen reader support

## Code Style

- Use 4 spaces for indentation
- Keep HTML semantic and accessible
- Comment complex JavaScript logic
- Follow existing naming conventions

## Questions?

Open an issue and we'll help you out!
